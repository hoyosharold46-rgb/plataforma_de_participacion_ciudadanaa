## ✅ Requisitos funcionales

El script muestra una ficha de datos de una propuesta ciudadana con:

### ✔ Tipos de datos usados

- `string`: nombre de la propuesta
- `number`: cantidad de votos
- `boolean`: estado de aprobación
- `null`: dato no asignado

---

### ✔ Validación de tipos

Se usa `typeof` para verificar los tipos de:

- nombre
- votos
- estado

---

### ✔ Conversión de datos

Se convierte un número a texto usando `String()`

---

### ✔ Nomenclatura

- Variables en `camelCase`
- Boolean con prefijo `is`
- Número con separador `_`
