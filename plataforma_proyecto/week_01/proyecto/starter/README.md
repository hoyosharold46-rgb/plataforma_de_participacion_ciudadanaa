## 🎯 Objetivo

Construir un script JavaScript que se ejecute en Node.js y muestre en la consola una **ficha de presentación** de una **propuesta ciudadana**, aplicando lo aprendido: `console.log()`, tipos primitivos (string, number, boolean) y comentarios.

No necesitas variables todavía. Solo `console.log()`, valores literales, expresiones y comentarios bien estructurados.

---

## 📋 Descripción

Vas a construir un "carnet digital de consola" para una **propuesta dentro de una plataforma de participación ciudadana**.

La propuesta representa una iniciativa de la comunidad (ej: mejorar un parque, crear ciclovías, seguridad barrial, etc.).

Debes mostrar información clara, organizada y relevante sobre esta propuesta.

---

## ✅ Requisitos

### Requisitos de contenido

El script debe mostrar **como mínimo**:

1. Nombre de la propuesta (string)
2. Código único de la propuesta (string o number)
3. Categoría o tipo de propuesta (string)
4. Al menos un dato numérico (ej: votos, meta, participantes)
5. Al menos dos valores boolean (ej: aprobada, visible, activa)
6. Una expresión aritmética (ej: total de votos, progreso)
7. Una sección de resumen del dominio

---

### Requisitos técnicos

- [ ] El archivo se llama `script.js` y corre con `node script.js`
- [ ] Usa `console.log()` para toda la salida
- [ ] Incluye mínimo **3 comentarios en español**
- [ ] La salida está organizada con separadores (`====` o `----`)
- [ ] No hay errores al ejecutar
